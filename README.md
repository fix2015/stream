# recorder video
Record video from timestamps
# Features
* SASS support including sourceMaps
* Minimal CSS styling of the view
* Gulp watch, build and local server tasks
* clear folder structure
* minified CSS and JS build files

## Structure of App
```
app/
    assets/
    js/
        /module
          /admin
            .js
          /recorder
            /components
                .js
            /service
                .js
        /service
            .js
        app.js
        config.js
    scss/
index.html
build/
```
